# LearnPerl

本项目是记录自己在学习Perl语言的过程中遇到的一些优秀内容，包括优秀的学习资源或优秀的项目代码等。尽可能的理解Perl，尽可能的理解Perl漏洞。学习Perl语言，面向无类型编程！作者：[0e0w](https://github.com/0e0w)
